# socialapp-backend

Backend of socialapp - application with filtering offensive post feature.
Feature is realized by separate application with neural network.

#### Technology stack:

* Java 8
* Spring Boot 1.5.9
* Spring Security 4.2.2
* Spring JDBC 4.2.2
* Guava 23.0
* Logback 1.2.3
* Groovy 2.4.7
* Spock 1.1
* Mockito 1.10.19
* MySQL

#### To run this app:

* Clone code
* Install maven
* ```mvn spring-boot:run```
